# Neural-Network-From-Scratch
Built a really simple neural network. Built for the purpose of understanding the fundamentals of neural networks and the math behind them.
This Neural Network has 2 inputs, a 2 neuron hidden layer, and a one neuron output layer. This was purely made using just numpy and no other outside libraries. I built this so that I could understand how neural networks really work, with the idea of building more complicated and in-depth neural networks in the future. This neural network and everything that I learned from building this was with the help of Victor Zhou's intro to neural networks article, from the build your own x repository.
Detailed Notes on the learning process was documented in Neural_Networks.md
